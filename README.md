# Liber Militum — 40K Army Tracker

A grimdark Warhammer 40K army painting tracker. Tracks units across six stages: On Sprue → Assembled → Painted → Based & Primed → Battle Ready (complete).

## Deploy to GitHub Pages

1. Create a new repo on GitHub (e.g. `liber-militum`)
2. Upload all files in this folder to the repo root
3. Go to **Settings → Pages → Source** and set it to `main` branch, `/ (root)`
4. Your app will be live at `https://yourusername.github.io/liber-militum`

## Add to home screen (iPhone / Android)

- Open the URL in Safari (iPhone) or Chrome (Android)
- Tap Share → **Add to Home Screen**
- Launches full-screen like a native app, works offline

## Icons

Replace `icon-192.png` and `icon-512.png` with your own artwork (a chapter icon works great) for a proper home screen icon. Free tools like https://realfavicongenerator.net can generate all sizes from one image.

## Data

Data is saved in `localStorage` per browser. Use **Export roster** to back up your data as JSON, and **Import roster** to restore it or move it to another browser.
